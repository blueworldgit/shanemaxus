1. Project Overview & Architecture

The goal was to transform the homepage and navigation of shane.maxusvanparts.co.uk to match the design and functionality of maxusvanparts.acstestweb.co.uk (the source/reference site), while keeping the existing WooCommerce database structure intact.

What Was Built

  • Homepage visual facelift (hero banner, vehicle carousel, Why Us section, custom footer)
  • Vehicle landing pages at /vehicle/{slug}/ with organic DB-driven category grids
  • Department pages at /department/{slug}/ aggregating vehicles by part category
  • VIN lookup (17-character VIN → vehicle identification)
  • Registration lookup (UK plate → vehicle identification via external API)
  • Replacement search bar (Model/Year/VIN/Registration)
  • Header dropdown lookup panels on nav menu items

What Was NOT Changed

  • No WooCommerce products, categories, or core data were modified
  • No existing database tables or structures were altered
  • No plugins were added or removed
  • The existing Mobex parent theme was not touched

Single-File Architecture

ALL custom code lives in ONE file:
wp-content/themes/mobex-child/functions.php

Lines 1–106: Original child theme code (parent style enqueue, theme mod repair, REST API endpoint)
Lines 107–2127: All facelift code (10 sections)

There are no additional plugins, templates, JS files, or CSS files. All custom CSS, HTML, PHP, and JavaScript is injected via WordPress hooks.

function mvp_vin_lookup() {
    $vin = isset( $_POST['vin'] ) ? sanitize_text_field( $_POST['vin'] ) : '';
    $vin = strtoupper( preg_replace( '/[^A-Za-z0-9]/', '', $vin ) );

    if ( strlen( $vin ) !== 17 ) {
        wp_send_json_error( array( 'error' => 'VIN must be exactly 17 characters' ) );
    }
    if ( substr( $vin, 0, 2 ) !== 'LS' ) {
        wp_send_json_error( array( 'error' => 'This does not appear to be a Maxus VIN (should start with LS)' ) );
    }

    // Model year from position 10 of VIN
    $year_codes = array(
        'A'=>2010,'B'=>2011,'C'=>2012,'D'=>2013,'E'=>2014,'F'=>2015,'G'=>2016,
        'H'=>2017,'J'=>2018,'K'=>2019,'L'=>2020,'M'=>2021,'N'=>2022,'P'=>2023,
        'R'=>2024,'S'=>2025,'T'=>2026,'V'=>2027,'W'=>2028,'X'=>2029,'Y'=>2030,
    );
    $customer_pattern = substr( $vin, 0, 8 );
    $customer_year_code = substr( $vin, 9, 1 );
    $customer_year = isset( $year_codes[ $customer_year_code ] ) ? $year_codes[ $customer_year_code ] : null;
    $home_url = home_url( '/' );
    $vehicles = mvp_get_vehicle_vins();

    // 1. Try exact VIN match first (VIN categories ARE full VINs)
    foreach ( $vehicles as $slug => $v ) {
        if ( strtoupper( $v['vin'] ) === $vin ) {
            wp_send_json_success( array(
                'vehicle_name'  => $v['name'],
                'customer_year' => $customer_year,
                'shop_url'      => $home_url . 'vehicle/' . $slug . '/',
            ) );
        }
    }

    // 2. Pattern match by first 8 chars of VIN
    $matches = array();
    foreach ( $vehicles as $slug => $v ) {
        $v_pattern = substr( strtoupper( $v['vin'] ), 0, 8 );
        if ( $v_pattern === $customer_pattern ) {
            $matches[ $slug ] = $v;
        }
    }

    if ( empty( $matches ) ) {
        wp_send_json_error( array(
            'error'            => 'No vehicle found for VIN pattern: ' . $customer_pattern,
            'suggestion'       => 'We may not have parts for this specific Maxus model yet.',
            'customer_pattern' => $customer_pattern,
            'customer_year'    => $customer_year,
        ) );
    }

    // Single match
    if ( count( $matches ) === 1 ) {
        $slug = array_key_first( $matches );
        $v = $matches[ $slug ];
        wp_send_json_success( array(
            'vehicle_name'  => $v['name'],
            'customer_year' => $customer_year,
            'shop_url'      => $home_url . 'vehicle/' . $slug . '/',
        ) );
    }

    // Multiple matches — narrow by year
    $best_slug = null;
    if ( $customer_year ) {
        foreach ( $matches as $slug => $v ) {
            if ( preg_match( '/(\d{4})\s*-\s*(\S+)/', $v['year'], $m ) ) {
                $start = intval( $m[1] );
                $end = ( $m[2] === 'Present' ) ? 2030 : intval( $m[2] );
                if ( $customer_year >= $start && $customer_year <= $end ) {
                    $best_slug = $slug;
                    break;
                }
            }
        }
    }
    if ( ! $best_slug ) $best_slug = array_key_first( $matches );

    $v = $matches[ $best_slug ];
    wp_send_json_success( array(
        'vehicle_name'  => $v['name'],
        'customer_year' => $customer_year,
        'shop_url'      => $home_url . 'vehicle/' . $best_slug . '/',
    ) );
}

    $fuel  = isset( $body['fuelType'] ) ? trim( $body['fuelType'] ) : '';

        ) );
    }

    // Match model to vehicle landing page
    $vehicles = mvp_get_vehicle_vins();
    $home_url = home_url( '/' );
    $best_slug = '';
    $best_name = '';
    $model_upper = strtoupper( $model_name );

    foreach ( $vehicles as $slug => $v ) {
        $v_name = strtoupper( $v['name'] );

        // Direct match
        if ( stripos( $model_upper, $v_name ) !== false || stripos( $v_name, $model_upper ) !== false ) {
            $best_slug = $slug;

            break;

        foreach ( $keywords as $kw ) {
            if ( stripos( $model_upper, $kw ) !== false && stripos( $v_name, $kw ) !== false ) {
                $is_electric = ( stripos( $fuel, 'ELECTRIC' ) !== false );
                $v_is_electric = ( stripos( $v_name, 'E DELIVER' ) !== false || stripos( $v_name, 'EV' ) !== false );
                if ( $is_electric === $v_is_electric ) {
                    $best_slug = $slug;
                    $best_name = $v['name'];
                    break 2;
                }
                if ( empty( $best_slug ) ) {
                    $best_slug = $slug;
                    $best_name = $v['name'];
                }
            }

17c. Duplicate SKUs Across VINs

The same physical part (same SKU) exists as separate WooCommerce products under different VIN categories. This creates scale concerns when adding brands like MG, and Google SEO issues where SKU searches could land on any VIN's version.

Recommendation: Consider single-product-per-SKU approach with vehicle compatibility attributes for multi-brand scaling.

17d. Vehicle Images Hosted on Source Site

All images reference https://maxusvanparts.acstestweb.co.uk/wp-content/uploads/. These should be copied to the production site's own uploads directory before go-live.

17e. Rewrite Rules Need Flushing After Deploy

WordPress stores rewrite rules in the database. If lost, re-flush via wp rewrite flush or flush_rewrite_rules().
